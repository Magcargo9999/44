### 문제 1. Stakeholder란 무엇이고, Stakeholder가 증가할 경우에 발생하는 문제점에 대해서 "추상화와 모델링"이 어떠한 효과가 있는지 설명하시오.

- **Stakeholder**는 하나의 프로젝트에 이해관계를 가진 **이해관계자**를 말합니다.
- 한 프로젝트에는 고객, 사용자, 기획자, 개발자 등 수많은 이해관계자가 존재합니다. 

- **문제점:** 이해관계자가 늘어날수록 소통 채널이 기하급수적으로 증가하여 원활한 소통이 어려워집니다.

- **추상화와 모델링의 효과**
    - **추상화:** 핵심적인 부분만 추출하여 단순화함으로써 이해를 돕습니다.
    - **모델링:** 설계 도면을 시각화하여 동일한 관점에서 소통하게 합니다.
